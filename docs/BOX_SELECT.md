# 建筑物框选功能

## 功能概述

支持在 3D 画布中使用框选功能选择多个建筑物，并进行批量操作。

## 使用方法

### 框选操作

1. **开始框选**：按住 `Alt` 键
2. **绘制选择框**：在画布上按住鼠标左键并拖拽（移动超过 5 像素）
3. **完成选择**：松开鼠标左键，框选范围内的所有建筑物将被选中

### 点选操作（Alt + 单击）

- **Alt + 单击建筑物**：切换该建筑物的选中状态
  - 如果未选中 → 添加到选中列表
  - 如果已选中 → 从选中列表移除
- 这样可以精确选择多个不相邻的建筑物

### 多选操作

- **组合使用**：可以混合使用框选和点选来精确选择建筑物
- **点击空白区域**：清除所有选中状态

### 批量操作

选中多个建筑物后，可以进行以下操作：

#### 移动
- 使用 `方向键`（上/下/左/右）移动所有选中的建筑物
- 每次移动 1 个单位

#### 复制
- `Ctrl+C` / `Cmd+C`：复制所有选中的建筑物
- `Ctrl+V` / `Cmd+V`：粘贴所有复制的建筑物
  - 粘贴的建筑物会自动偏移，避免重叠
  - 连续粘贴会累加偏移量

#### 删除
- `Delete` 或 `Backspace`：删除所有选中的建筑物

## 视觉反馈

### 单选高亮
- 白色边框线框

### 多选高亮
- 绿色边框线框（#52c41a）
- 半透明效果（85% 不透明度）

### 框选矩形
- 蓝色边框线框（#1677ff）
- 实时显示选择范围

## 注意事项

1. **智能识别单击/拖拽**：按住 Alt 键时，系统会智能区分单击和拖拽操作
   - 鼠标移动 < 5 像素：识别为单击，触发点选
   - 鼠标移动 ≥ 5 像素：识别为拖拽，触发框选
2. **框选时禁用相机控制**：在框选过程中，OrbitControls 会被自动禁用，避免相机旋转
3. **松开 Alt 键取消**：如果在拖拽过程中松开 Alt 键，框选将被取消
4. **点击空白区域清除选择**：点击地面会清除所有选中状态
5. **多选优先**：当同时存在单选和多选时，键盘操作会作用于所有选中的建筑物

## 技术实现

### 状态管理
- `selectedBuildingId`：单个选中的建筑物 ID（用于兼容旧代码）
- `selectedBuildingIds`：多个选中的建筑物 ID 数组

### 关键组件
- `SelectionBox`：显示框选范围的 3D 矩形
- `BoxSelectInteraction`：处理 Alt+鼠标拖拽事件
- `BuildingMesh` / `GlbBuildingMesh`：支持多选高亮显示

### 坐标转换
使用 Raycaster 将屏幕坐标转换为 3D 地面坐标（y=0 平面），确保框选范围准确。
